FONSAGUA 1.0

Descripción:

Este desarrollo son varias extensiones que personalizan y simplifican gvSIG para el uso de Información Geográfica en el proyectos de cooperación FONSAGUA (Agua en el Golfo de Fonseca) de Ingeniería Sin Fronteras Galicia, Save de Children Honduras, Alcaldía de Marcovia, Comité para Defensa y Desarrollo de la Flora y Fauna del Golfo de Fonseca

Características:

  • Extensión para la configuración de la carpeta de datos
  • Extensión para la carga automática de los datos del proyecto
  • Extensión de consultas para la exportación de datos clave a formato de ficheros csv
  • Extensión para la creación de informes en formato rtf
  • Extensión para la copia de los datos procedentes de un GPS en las capas del proyecto
  • Personalización de NavTable para la introducción y consulta de datos mediante formularios específicos
  • Personalización de NavTable para el guardado dual sobre shapefiles y base de datos sqlite
  • Panel base de lanzamiento de formularios de consulta e introducción de datos
